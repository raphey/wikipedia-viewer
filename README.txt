This Wikipedia API tool was created as a project for FreeCodeCamp’s front-end development course.

It can be viewed at:
http://raphey.com/projects/wikipedia-viewer/index.html

Some features:
-Retrieves articles containing a search term using Wikipedia’s API
-Cycles through random articles roulette style
-Uses promises for more readable asynchronous function
-Pre-caches the random articles for the roulette wheel so it works right away
-Responsive design courtesy of Bootstrap plus some customization
